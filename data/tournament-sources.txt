Causey — tournament data sources
================================
Formal APIs for live/upcoming OTB calendars barely exist. Causey indexes
public tournament hubs via scrapers (TypeScript + Cheerio), not organizer APIs.

Status key:
  LIVE     — scraper shipping; events may appear on /chess
  SOON     — planned; not wired yet
  AFFILIATE — state calendars; full list in data/state-affiliates.txt

-------------------------------------------------------------------------------
LIVE (scraping now)
-------------------------------------------------------------------------------
US Chess Federation — Tournament Life Announcements (TLA)
  https://new.uschess.org/upcoming-tournaments
  Official directory of USCF-rated events. Primary US supply feed.
  Pipeline: source = tla_scrape  |  npm run scrape:tla

Continental Chess Association (CCA)
  https://www.chesstour.com/refs.html
  Major US opens (World Open, National Chess Congress, etc.).
  Pipeline: source = cca_scrape  |  npm run scrape:cca

-------------------------------------------------------------------------------
SOON (adding next — national / registration hubs)
-------------------------------------------------------------------------------
OnlineRegistration.cc
  https://onlineregistration.cc
  Organizer registration hub used by many US events.

Chess-Results.com
  https://chess-results.com
  Global pairings/results hub (Swiss-Manager publishes here).
  Target: Tournament Search for upcoming USA / FIDE events.

FIDE Calendar
  https://fide.com/calendar
  Official international events (World Cup, Candidates, Grand Swiss, etc.).
